EXTRACTION RULES:
1. Extract every table as a JSON array of row-objects.

2. JSON key  =  EXACT column header name. Preserve:
     • Original case              pH, BOD5, TSS, NH4-N, Cu2+
     • Units inside header        "Temp (°C)", "Conc. (mg/L)", "Flow (m³/day)"
     • Chemical notation          SO4 2-, NO3-, Fe2+, Cl-
     • Range notation             "6.5 – 8.5", "80–120 %"
     • Superscripts / subscripts  write inline (e.g. m³, µg/L, cm²)

3. Extract ALL rows in EXACT document order. Never skip, merge, or reorder.

4. Cell value rules:
     Numeric              → keep exact as written, e.g. 10.52, 0.003
     Below detection      → keep verbatim: <0.01  ND  BDL  <LOD  <LOQ
     Uncertainty / range  → keep verbatim: 10.5 ± 0.3   10.5–11.2
     Pass / Fail / status → keep as string: "Pass"  "FAIL"  "Compliant"
     Empty cell           → null
     Text note / footnote → keep as string

5. Multi-row column headers → flatten to one string joined with " – "
   Example: "Concentration" over "mg/L" → "Concentration – mg/L"

6. Multiple tables → assign key from table title / caption (verbatim).
   If no title: "Table_1", "Table_2", etc.
